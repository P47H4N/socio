# Socio - A Social Media

Socio is a quality-focused social media platform featuring a robust **Golang** backend, a dynamic **React** web interface, and a high-performance **Flutter** mobile application. Built for speed, scalability, and security, Socio provides a seamless real-time experience across all devices.

## 🚀 Technology Stack

* **Backend:** Go (Golang)
* **Frontend:** React JS
* **Mobile App:** Flutter
* **Database:** PostgreSQL
* **Authentication:** JWT (JSON Web Tokens) & Bcrypt (Secure Hashing)
* **Real-time:** WebSockets
